Web

처음으로 진행한 팀프로젝트입니다. 전국의 사진찍기 좋은 명소를 소개하는 사이트로, 카테고리별로 사진을 볼 수 있고, 원하는 지역을 선택하여 해당 지역의 사진만 볼 수도 있습니다. 제가 담당한 부분은 html과 script 부분이였습니다.

AJAX를 이용한 비동기 처리를 처음 접해본 프로젝트였고 스크립트 코드를 여러 번 다시 짜보면서 AJAX에 대한 이해를 넓히기 좋았습니다. 처음에는 xml 파일을 이용하여 데이터를 불러왔지만 xml 파일을 json 파일로 바꾸면서 json 파일을 이용하여 비동기 처리를 했습니다.

이 프로젝트의 스크립트 파일은 총 세 번 작성되었습니다. 처음에는 jQuery를 이용하여 제작하였고 두 번째는 Vanilla Script를 이용하여 제작하였습니다. 세 번째는 최적화를 위해 코드를 다시 작성하였습니다.

처음에는 모든 사진을 불러온 뒤 한 장씩 보여주는 방식이었다면 마지막에 작성한 코드에서는 사진이 바뀔 때만 필요한 데이터를 불러오는 식으로 바꾸었고, 사이트 유지보수를 쉽게 만들기 위해 데이터 수정을 할 영역을 최소화하는 쪽으로 바꾸려는 노력을 했습니다.
